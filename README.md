# SCT_CS_2

## Simple Image Encryption using Pixel Manipulation

This project demonstrates a basic image encryption and decryption technique
using pixel manipulation in Python.

The encryption is done by:
- Swapping pixel color values
- Applying a mathematical operation using a key

The process is reversible, allowing decryption.

---

## Features
- Encrypt image using pixel swapping
- Apply mathematical operation on pixel values
- Decrypt image using the same key
- Simple and easy to understand

---

## Requirements
- Python
- Pillow library

Install Pillow using:
pip install pillow

---

## How to Run
1. Place an image named `input.png` in the project folder
2. Run:
python image_encrypt.py
3. Choose encrypt or decrypt
4. Enter the key value

---

## Output Files
- encrypted.png
- decrypted.png
